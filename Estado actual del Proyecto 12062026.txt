# SmartWMS 2.0 – Estado Actual del Proyecto (Resumen para Retomar en Codex)

## Objetivo del Proyecto

SmartWMS 2.0 es un Warehouse Management System desarrollado sobre .NET 8, SAP Business One y arquitectura modular, orientado a operaciones reales de almacén con:

* Recepción
* Inventarios
* Movimientos
* Picking
* Packing
* Shipping
* Integración SAP B1
* Dispositivos móviles Zebra
* Auditoría
* Dashboard operativo

El objetivo actual es completar el MVP Productivo y posteriormente evolucionar hacia SmartWMS 3.0.

### Problema actual

Al abrir Login:

```text
Unhandled exception on current circuit
Circuit terminated
```

Error reportado:

```text
blazor.web.js

There was an unhandled exception
on the current circuit.
```

Esto indica un error en servidor Blazor.

Debe activarse:

```json
{
  "DetailedErrors": true
}
```

en:

```text
appsettings.Development.json
```

o

```csharp
builder.Services.AddServerSideBlazor()
    .AddCircuitOptions(x =>
    {
        x.DetailedErrors = true;
    });
```

para identificar la excepción real.
